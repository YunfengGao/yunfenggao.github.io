《焦虑的人》

发表于 2025-05-05

简介

究竟是何种程度的无奈和绝望，迫使一个中年人在新年来临前的早晨，用一把玩具枪抢劫一家无现金银行。行动失败的劫匪仓皇之中逃进一间位于大楼顶层的待售公寓，里面全都是正在看房子的潜在买家。由此开始，抢劫案变成了劫持人质案。

然而事情的发展出乎预料，没有警匪对峙、没有致命一击，漫长的一天过后，八名人质安然无恙得到释放，劫匪却完全不见踪影。警察对人质逐一展开讯问，却发现他们每个人都有一肚子的抱怨和疑问，可是谁也说不清楚并且不关心劫匪究竟去了哪里。

婚姻、工作、家庭、育儿、投资、医疗、人际关系，现如今的生活，任何事情都会引发焦虑，没有人敢说自己一点儿都不焦虑。我们都是辛苦又脆弱的小人物，一边吐槽，一边还在咬牙坚持过下去。小说《焦虑的人》关照的正是平凡的你我。

摘抄

因为步入成年的可怕之处在于，我们会被迫意识到，绝对没人有人在乎我们。我们必须亲自处理所有事情，弄明白世界是如何运作的。工作、支付账单、用牙线、按时开会、规规矩矩排队、填各种表格、整理数据线和电源线、组装家具、换车胎、给手机充电、关咖啡机，还有，别忘记给孩子报游泳课。 早晨一睁开眼，各种“别忘记了”和“记住了”就如同雪崩，排山倒海而来，我们没有时间思考和呼吸，只能在“雪堆”里挖出一条活路，而第二天又会出现新的一堆。

房地产市场会再一次崩溃，就算不在明天，那也是早晚的事。 作为银行，我们很清楚这一点，可我们还是往外借钱，当借钱的人失去一切的时候，我们就说这是他们的责任，这是游戏规则，全部是他们自己的错，因为他们太贪婪了。可这当然不是事实，大部分人并不是贪婪，而是……就像我们讨论这幅画的时候你说的那样：他们需要找到可以抓住的东西，可以为之争取的东西——他们只是想要有个住的地方，在那里抚养孩子和过自己的生活。

可你是否曾静恋爱过？几乎每个人都有过这样的经历。爱情会让你做出很多荒唐事，比如结婚、生孩子、扮演幸福的一家人，拥有幸福的婚姻。你也有可能觉得不快乐，但这并不稀奇，在婚姻里面很常见，因为一个人不可能一直都开心，开心的程度也参差不齐，而且有时候根本没时间开心。通常情况下，我们只是在过日子而已。你的生活或许也是这样，可你也会有受够了的时候。

当然，事实上，如果人们真的像他们在网上表现的那么开心，就不会把那么多该死的时间花在互联网上了，因为没有谁会在真正度过愉快的一天时，还能想着拿出半天的时间自拍。只要拥有足够的肥料，任何人都可以给生活营造虚假的光鲜氛围，所以，假如对面邻居家的草坪看起来更绿，很可能是因为那边的狗屎更多。我们现在知道，每一天都应该是特别的，当然，区别不用那么大，关键在于每天都不重样。

你能想象吗，你的孩子不想要孩子，这说明你们得是一对多么糟糕的父母啊！

孩子会吸走家里所有的氧气，挤占成年人情感交流的空间，有些家长设置一连很多年都找不到机会向别人倾诉自己的感受，加入你长久以来一直无法获得这样的机会，有时候就会完全忘记该怎么做。

跟十多岁的孩子一起住过的人都知道，他们只为自己而活，青少年家庭的服务和子女多半都在忙着应付人生中最棘手的难题，这样的环境当然没有艾斯特尔的位置，大部分情况下，她都是个令人讨厌的存在。他们只会在她过生日时高高兴兴地打来问候电话，其余的时候就把她当作一件不受岁月影响的漂亮装饰，只在圣诞节和仲夏节的时候拿出来摆摆样子。

有时候我觉得，假如你们在一起生活很长时间，而且又有孩子的话，生活就有点像不停地爬树，上去再下来，上去再下来，几乎顾不上看看对方。年轻的时候你是不会注意到这些的，但当你有了孩子，一切都会改变，有时候你甚至完全看不见跟你结婚的那个人。

小的时候，你渴望长大，自己决定一切，可是一旦长大，你就会意识到，“做决定”是成年人所要面对的最可怕的任务。你心里必须一直有主意，才能及时地拿定主意。你得知道应该给哪个党派投票、自己喜欢什么样的墙纸、你喜欢什么样的伴侣、哪种口味的酸奶最能体现你的个性。你必须做出选择，同时也被别人选择。这样的事几乎每时每刻都发生。

酒鬼擅长说谎，但他们的孩子更擅长，因为酒鬼的子女往往不得不找借口掩盖某些事实，想出的理由不能过于古怪或者令人难以置信，必须稀松平常到让人懒得怀疑。比如，酒鬼的孩子的作业本永远不会被狗吃掉，他们只不过是把书包忘在了家里；他们的妈妈没来开家长会，绝对不会是因为被忍者绑架了，而是必须加班，孩子不记得她在哪里工作，因为她只是个临时工；爸爸走了，是妈妈在尽全力养活这个家，你知道的。

吉姆自然会尽最大努力展现自己富有经验，想想父亲们是多么喜欢教儿子做事就知道了。因为一旦我们没有什么可以教给孩子，就意味着从那一刻开始，他们不再是我们的责任，我们反而成了他们的责任。

【转载】劳动者驯化

发表于 2021-05-26

劳动者的驯化是一个复杂的系统性的工程，资本家是推动这个进程的最大收益者，但在一些资本主义国家zf也是这个进程的参与者。

首先是从教育上

第一步就是需要筛选出哪些是未来的「劳动者」，在这方面美国教育制度的优越性可是很明显的。通过入学，考试，等手段筛选出潜在的「劳动者」，至于那些不是劳动者的当然是去学习对他们真正有用的东西，培养高尚的情趣和优雅的气质了。这里不做讨论。
在分层完成后，教授潜在劳动者劳动所需技能，这也不多说，但需要注意的是，劳动者拥有的技能越是单一实际上对资本家是越好的。
更重要的是对认知的塑造，首先就是教授内容的选择性，告诉你比尔盖茨是怎么肄业发家的，告诉你福特公司是怎么做流水线提高生产效率的，但不会告诉你比尔盖茨的家庭背景，也不告诉你福特公司在提高生产效率的同时怎么压榨工人的。这一切都是为了提升你对现存制度（资本主义）的好感度。仿佛每个人都能在此制度下获得美好的人生（保证你在结婚生子背上生活压力前不搞事情）
当劳动者比较大了，从媒体上获取信息的时候，就需要资本主义造神，由资本家控制的媒体选出一个杰出资本家代表，首先宣扬奋斗和梦想，改变世界，让未来更好什么的。年轻人很容易受此蛊惑。奋斗学就是典型代表，我个人从不否认奋斗的意义，但我对给资本家打工奋斗的意义表示怀疑。一直到这里，还是在用鸡汤来保证稳定，但也要开始下一步的布局了。
等潜在的劳动者再大大，有一部分感觉出不对劲来了，或者一部分对成功学，奋斗学没啥感觉了，就要祭出另一手段。

消费主义

消费主义百度对此释义的十分透彻和赤裸裸，「消费本身已经不视作一种手段，而是成为目的本身。」
通过各种手段，来制造需求，我本人没有性别歧视，但不得不说，女性是消费主义受害比较深的群体，而且效果比较好的也是女性群体（这里涉及到女性和繁衍权的关系，繁衍权与两性关系等。不做展开）人工制造需求，比如时尚快消品，比如挑动或者物化两性关系，来促进消费或者男性对女性的消费，这里需要说明，促进适龄人消费，无论男性女性，实际上就是减少未来的家庭存款。
这里展开说一下，消费主义本身，它对于资本主义国家是十分必要的，1-它拉动需求，2-让一个人老老实实待在工厂里最好的办法就是让他需要钱。
当消费主义把年轻人的钱包掏空之后，年轻人就必须去打工，而且消费主义很妙的一点是它给人一种生活质量很高的错觉，这对于年轻人来说很重要，况且年轻人养家压力很小甚至没有，让年轻人老老实实上班就需要让他们增加开销。
然后下一步就是让劳动者背上贷款，我很久之前在一个关于房价下的问题里讲过，贷款是社会安定的一个重要手段。同理贷款是让劳动者老老实实在岗工作甚至接受高强度加班的一个重要手段。

最后一步，通过医疗，和教育对劳动者家庭剩余存款的剥夺。

这里最有效的手段是教育。因为有医疗保险的存在。虽然也有事实上的教育保险，但教育费用实际上是递增的。这里还涉及到中产阶级的一个问题，首先明确中产阶级实际上是无产阶级。这个读过马克思的人都能理解。对于中产阶级家庭来说，略夺家庭财富的就是教育。因为中产阶级比城市无产阶级过得日子要好，他们的存款或者可支配收入很多会用于培养下一代，并且中产阶级普遍希望孩子能实现阶级上升。这时候一批培训机构，一批贵族学校会来吃掉中产阶级家庭多余的钱，可我们能看到，中产阶级和资产阶级的壁垒有难打破。而且随着资本主义的发展，阶级壁垒越来越强。到时候光是让孩子维持中产阶级的水平就已经很难了。至于普通城市无产阶级更不用说，他们本来就剩不下什么，家里孩子老师等着用钱，不工作咋办？加班也得硬着头皮上。
到了这一步，资本家获得了，可以接受加班，但无法承受失业的劳动者。
为了获得这种被驯化完成的劳动者先后进行几项操作

  1. 通过教育让人认同资本主义国家的制度，并相信每人都有机会。这一步有两个目的，首先是让人在年轻时不搞事，其次对制度的认同也方便之后的驯化。
  2. 通过媒体宣传，建立起一种有利于资本家的思想观念，就像让古埃及的奴隶相信努力推石头自己有一天也能拥有金字塔一样（只是打个比方，当时奴隶只用鞭子就能驯服，但随着参与劳动个体的不断独立化精神上的控制更加重要且有效了）这一步目的和上一步类似。但比上一步从逻辑上更进一步，这里强调资本家的正面形象了。
  3. 消费主义，通过各种方式让人消费，并假装自己生活的不错。甚至会生出一种阶级认知的错觉。这会让年轻人老实待在工作岗位上勤奋工作。
  4. 贷款，通过贷款让劳动者背上生活的压力，有了压力不一定有动力，但有了压力普遍胆小。到这一步已经不需要什么宣传和思维上的控制了。
  5. 高价医疗和教育对家庭的最后压榨。贷款还完了，孩子要上学的嘛，还要防一手自己生病嘛，这里需要指出的是，医疗这个方式不一定需要把钱真正的拿走，只需要让普通家庭一般不敢动这个钱就行，或者通过医疗保险拿走其中一部分。
    所有这一切你搞定了，劳动者无欲无求了。也就差不多该退休了。

我觉得最后需要补充几个点

  1. 这些操作肯定不是某一个资本家能完成的，但在资本主义的市场经济下，自发行程了不同的驯化阶段，这每一个阶段都是资本家获利，反过来这些操作也巩固了资本主义。
  2. 以上基本以中产家庭作为假设对象。对普通无产者的驯化更加粗暴，既只要基础工资足够低就不愁“主动”加班。形成这种差异是由于中产阶级虽然属于无产阶级但更多从事文职，脑力劳动。而非传统意义上的工厂劳动。而且劳动技能更加多样化，并且对生产工具的依赖往往没有工业类的那么严重，相对来说他们的附属性更低，所以需要报酬更多，并且他们抚养下一代标准也高。但无产阶级的基本属性没变。
  3. 以上内容以资本主义国家作为背景。

三、气质类型

根据大卫.凯尔西（David Keirsey）气质与性情理论，你属于“理想主义者”，下面是对“理想主义者”的描述：

“理想主义者”型的人感兴趣的是事物的意义、关系和可能性，并基于其个人的价值观念做出决定。这是一类关心个人成长和如何理 解他人与自我的人。

“理想主义者”做人的原则是：“真实地面对自已”，是精神上最具哲理性的人。“理想主义者”乐于接受新的思想，善 于容纳他人。

“理想主义者”好象永远在寻找生存的意义。他们非常崇尚人与人之间和各种关系中的真实和正直，容易将别人理想化。许多“理想 主义者”本能地喜欢帮助别人成长和进步。“理想主义者”是很好的传播者，被人们认为是促进积极变化的催化剂。

他们天生能够理解别人的情感，关心他们在生活、工作中碰到的人们（如同事、病人或客户、雇员）的需要。

对于“理想主义者”型的人而言，一份好的工作应该是对他们个人很有意义的工作，而不是简单的常规工作或只是一种谋生手段。“ 理想主义者”崇尚和谐，不愿意在一种竞争激烈或四分五裂的环境中发展。他们喜欢民主、能够激励各种层次的人们高度参与的组织 。他们会被那些促进人性价值的组织或那些允许他们帮助别人完成工作的职业所吸引。

总体描述

  1. 对别人的情绪敏感，能理解、体会别人的心情，善于安慰、鼓励别人
  2. 对文字、语言敏感
  3. 善于分析、总结
  4. 善于从整体上把握事物
  5. 能理解复杂的理论概念，善于将事情概念化，善于从中推断出原则
  6. 擅长策略性思维

潜在的弱点

  1. 有仅仅凭个人的好恶或价值观来决定事情，并希望别人也以同样的角度或标准来处理问题的倾向。
  2. 有时他们心里老想着别人的问题，可能会过于陷于其中，以至于被其困扰。
  3. 有时容易将别人或事情理想化，不够实际。
  4. 不是特别善于管束和批评他人，尽管常常自我批评。有时会为了和睦而牺牲自己的意见或利益。
  5. 有些此类型成员比较容易动感情，情绪波动较大。
